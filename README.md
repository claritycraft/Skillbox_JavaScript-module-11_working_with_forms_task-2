### En:
#### Task 2
Create a web page to calculate the delivery cost of products.
Each product has the following characteristics:
- Product Name: The user enters the name of the product.
- Product Weight (in kilograms): The user specifies the weight of the product in kilograms.
- Delivery Distance (in kilometers): The user specifies the delivery distance in kilometers.
 Steps to follow:
- Create an HTML form that allows the user to enter product data (name, weight, and delivery distance) and add the product to a list for delivery cost calculation.
- Implement data validation to ensure the entered values are correct (positive numbers, proper product name format).
- After successful validation, add the product to the list showing its name, weight, and delivery distance. Be sure to clear the form for new entries.
- Calculate the delivery cost for each product using the following formula:
delivery cost = (product weight * delivery distance) / 10
- Display the list of added products, including name, weight, delivery distance, and calculated delivery cost.
- Style the page and the table for a more attractive presentation.

### Ru:
#### Задача 2
Создайте веб-страницу для калькуляции стоимости доставки товаров. Каждый товар имеет следующие характеристики:

Название товара: пользователь вводит название товара.
Вес товара в килограммах: пользователь указывает вес товара в килограммах.
Расстояние доставки в километрах: пользователь указывает расстояние доставки в километрах.

Последовательность действий:

Создайте HTML-форму, которая позволяет пользователю вводить данные о товаре (название, вес и расстояние доставки) и добавлять этот товар в список для расчёта стоимости доставки.
Реализуйте валидацию данных, введённых пользователем, чтобы гарантировать корректность введённых значений (числа положительные, формат названия товара правильный).
После успешной валидации формы добавьте товар в список с указанием названия, веса и расстояния доставки. Не забудьте выполнить очистку формы для ввода новых параметров.
Рассчитайте стоимость доставки для каждого товара по следующей формуле:
стоимость доставки = (вес товара * расстояние доставки)/10
Отобразите список добавленных товаров с указанием названия, веса, расстояния доставки и стоимости доставки.
Оформите стили для страницы и таблицы для более привлекательного отображения.
